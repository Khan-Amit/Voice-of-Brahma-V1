# scalePRT – Room-Temperature Planck Length Scale

## Definition

scalePRT is the fundamental measurement unit derived from the resonance curve of a U-bottom groove. It represents the effective length scale at which information is stored in the frequency domain.

## Interpretation

scalePRT is the **groove coupling length** – the distance over which two U-bottom grooves interact via resonance. It is not Planck-scale; it is a macroscopic length derived from resonance physics.

## Application

Used to calculate:
- Data density scaling
- Energy consumption per bit
- Throughput estimates
