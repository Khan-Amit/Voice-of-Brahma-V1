## What is This?

Voice-of-Brahma-V1 is the first complete implementation of a **ternary‑based universal computing language**. It translates any binary data (ASCII, HTML, C, Linux binaries) into ternary and back, using a 5‑trit ASCII encoding and an Enigma‑style rotor translator.

The platform is built on the **scalePRT** principle – a room‑temperature measurement of Planck‑scale energy dissipation, derived from the resonance curve of a U‑bottom optical groove.

## Core Components

- **Ternary ASCII Library**: 5‑trit encoding for all 128 ASCII characters.
- **Translator**: Enigma‑style rotors for binary ↔ ternary conversion.
- **Ternary Virtual Machine (TVM)**: Executes ternary instructions on any CPU.
- **scalePRT Calculator**: Computes the fundamental measurement unit from resonance width.
